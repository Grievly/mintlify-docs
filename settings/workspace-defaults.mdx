Each facility has its own default grievance SLA length.

Members can view the setting for facilities they can access. Only administrators can change facility SLA settings.

## Default facility SLA

The default grievance SLA sets the due date window for new grievances created at that facility.

The value is measured in hours from the received time.

1. Open **Facilities**.
2. Select the facility.
3. Open **Configuration**.
4. In **Grievance SLA**, enter the default SLA length in hours.
5. Save changes.

## Allowed range

The SLA must be at least 1 hour and no more than 720 hours.

720 hours equals 30 days.

## What changes affect

Changing a facility's default SLA affects new grievances created for that facility after the change.

It does not rewrite the history of already-created grievances.

Facilities can use different SLA lengths when their operational expectations differ.
