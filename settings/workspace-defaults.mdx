---

Workspace defaults control behavior for new records across the organization.

Only administrators can change workspace defaults.

## Default grievance SLA

The default grievance SLA sets the due date window for new grievances.

The value is measured in hours from the received time.

1. Open **Settings**.
2. Select **Grievance defaults**.
3. Enter the default SLA length in hours.
4. Save changes.

## Allowed range

The SLA must be at least 1 hour and no more than 720 hours.

720 hours equals 30 days.

## What changes affect

Changing the default SLA affects new grievances created after the change.

It does not rewrite the history of already-created grievances.
