---
title: "Attachments"
description: "Upload, scan, review, download, and manage supporting files on a grievance."
---

Attachments are supporting files that belong with a grievance record.

Supported uploads are PDF, Excel (`.xlsx`), Word (`.docx`), JPEG, PNG, WebP, CSV, and plain text files up to 5 MB each.

## Add an attachment

1. Open the grievance.
2. Find **Attachments** in the right rail.
3. Drag in a file or select **Upload attachment**.
4. Wait for upload and security scanning to complete.

Attachments also appear in the timeline so reviewers can see when they were added.

## Scan status

Uploaded files go through security scanning before they can be opened or downloaded.

Common states include:

- **Pending**: the scan is queued.
- **Scanning**: the scan is running.
- **Clean**: the file can be opened or downloaded.
- **Flagged**: the scan found a concern.
- **Failed**: the scan did not complete.

Only a clean attachment is available to open or download.

## AI summary

Supported attachments may show an advisory **Summary** panel on the timeline attachment card.

If CareResolve shows **Attachment may not match this concern**, the full summary stays collapsed behind **Show full summary**. Treat the warning as a review prompt and review the actual file before relying on either the summary or warning.

## Open or download an attachment

When a file has passed scanning, the timeline attachment card includes **Open** and **Download** actions.

Use **Open** for quick review in the browser. Use **Download** when the file needs to be saved for approved internal review or reporting.

## Locked and resolved cases

- While resolution approval is pending, users with case access can add an attachment, but existing attachments cannot be deleted.
- After resolution, only administrators can add an attachment.
- Existing attachments remain read-only after resolution, including attachments added by an administrator after closure.

## Delete an attachment

On an unlocked open grievance, use the attachment action menu and choose **Delete attachment**.

The file is removed from the active record and the deletion remains visible as timeline and audit activity. Attachments cannot be deleted while approval is pending or after resolution.

## Find grievances with files

The grievance list displays a paperclip and attachment count. Use **Other > Has attachments** to filter the list.

## Attachment safety

Only upload files that belong with the grievance. Reports and attachment packages can contain sensitive resident, reporter, or case information, so share them only with approved people.
